Portifólio Digital desenvolvido durante o Bootcamp I. Ciência de Dados e Machine Learning Olá! Seja bem-vindo ao meu repositório dedicado a Data Science e Aprendizado de Máquina. Este espaço documenta a minha trajetória na área tech, mesclando os projetos acadêmicos que realizo no CEUB com pesquisas autônomas focadas em automação e inteligência de dados.

Sobre o Portfólio A proposta central deste HUB é exibir habilidades práticas aplicadas no uso de Git para controle de versão, tratamento de dados utilizando a linguagem Python e o desenvolvimento de interfaces web estáticas.

Propósitos do Projeto Este ambiente foi estruturado para comprovar a minha proficiência nas seguintes áreas:

Engenharia de Dados e ETL: Higienização, estruturação e transformação de grandes volumes de informações.

Lógica de Programação: Criação de algoritmos eficientes e rotinas de automação.

Documentação Técnica: Elaboração de materiais descritivos claros, organizados e focados em processos.

Estrutura do Repositório O conteúdo foi dividido em módulos baseados em competências para otimizar a sua navegação:

🛠️ Ecossistema de Projetos Neste tópico, concentro as aplicações reais que construí. O objetivo aqui é ir além da teoria e mostrar a execução prática. Você encontrará desde simples scripts de automação até pipelines de dados mais complexos.

📊 Apresentações e Soft Skills Saber contar histórias com dados (Data Storytelling) é uma habilidade fundamental. Por isso, compartilho nesta área os materiais visuais e apresentações executivas que criei para defender os meus trabalhos. Eles refletem a minha aptidão para converter jargões técnicos em resultados de negócio com valor e impacto estratégico.

Meu Toolkit Tecnológico Linguagem Principal: Python (com ênfase em computação científica utilizando o ambiente Spyder).

Ciência de Dados: Tratamento e análise avançada com o uso das bibliotecas Pandas e NumPy.
